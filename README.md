## Краткое описание каждого модуля и процесса их выполнения
В первом модуле производятся работы по проектированию сетевой инфраструктуры. 

Второй модуль включает в себя работы по организации сетевого администрирования. 

Третий модуль посвящён эксплуатации объектов сетевой инфраструктуры.

Четвёртый модуль как правило разрабатывается либо самим учебным заведением, либо работадателями заинтересованными в рабочих кадрах.

Выполнение 1 модуля производится путем настройки каждой машины поочерёдно. Каждая машина настраивается в соответсвии с целями всех заданий демонстрационного экзамена

Выполнение 2 и 3 модуля производиться по ходу выполнения каждого задания, но зависит от вашего желания. Если какое-то задание кажется сложным вы можете пропустить его и приступить позже, чтобы не терять драгоценные минуты.

> [!Important]
> Максимальный балл за весь экзамен - 100 баллов. Каждый из модулей - 25 баллов.
