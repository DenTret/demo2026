**Решение выполнено благодаря добрым людям на просторах Github : [meowehh](https://github.com/meowehh/DemoExam_2026?tab=readme-ov-file), [shiraorie](https://github.com/shiraorie/demo2026-1?tab=readme-ov-file), а также долгому изуению различной документации.**

## Технические характеристики
Данный репозиторий предоставляет полную инструкцию по выполнению демонстрационного экзамена 2026 года профильного уровня - ПУ). Она включает в себя полное решение первых трёх модулей, а также пример отчётов по их выполнению.
> [!NOTE]
> Используется вариант стенда с роутерами на OC Ecorouter, серверами на OC Alt Server и рабочей станцией на OC Alt Workstation.

> [!Important]
> Скрипт развертки на PVE для Proxmox взят у [PavelAF](https://github.com/PavelAF/PVE-ASDaC-BASH).
