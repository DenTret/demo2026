**Шаг 7: Создание резервных копий**
```bash
sudo mkdir -p /var/backup/postgres
sudo chown postgres:postgres /var/backup/postgres
sudo -u postgres pg_dump postgredb > /var/backup/pgbackup_$(date +%Y%m%d)_1.sql
sudo -u postgres pg_dumpall --globals-only > /var/backup/pgbackup_$(date +%Y%m%d)_2.sql
```

**Шаг 8: Настройка автоматического резервного копирования**
**Создаём скрипт:**
```bahs
mcedit /usr/local/bin/postgres_backup.sh
```
```bash
#!/bin/bash
BACKUP_DIR="/var/backup/postgres"
DATE=$(date +%Y%m%d_%H%M%S)

sudo -u postgres pg_dump postgredb > ${BACKUP_DIR}/pgbackup_${DATE}_1.sql

sudo -u postgres pg_dumpall --globals-only > ${BACKUP_DIR}/pgbackup_${DATE}_2.sql

find ${BACKUP_DIR} -name "pgbackup_*" -mtime +7 -delete
```

**Далее делаем скрипт исполняемым**
```bash
sudo chmod +x /usr/local/bin/postgres_backup.sh
```

**Шаг 9: Настройка cron**
```bash

```
```bash

 ```
